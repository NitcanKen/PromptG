# 自用 Prompt 視覺化素材工作台｜產品需求文檔

## 1. 產品一句話定義

這是一個 **只服務於 Prompt 編寫與管理** 的自用工具。

它不負責生成圖片，也不接入任何 AI 圖像模型。它的核心任務是：

> 把一條完整圖片 Prompt 拆成很多個可視化、可分類、可點選、可重組的「Prompt 素材」，再讓使用者像搭積木一樣，用看圖選擇的方式組合出新的完整 Prompt。

使用者不是從零寫 Prompt，而是：

1. 先建立自己的素材庫。
2. 每個素材都是一段可複用 Prompt。
3. 每個素材都有圖片預覽、分類、標題、副標題、標籤和文字內容。
4. 使用時按分類看圖挑選素材。
5. 系統自動把已選素材編譯成一條完整 Prompt。
6. 使用者可以複製、修改、保存這條 Prompt。
7. 好的完整 Prompt 可以保存進自用 Gallery。
8. 好的完整 Prompt 也可以再次拆解，變成更多可重用素材。

---

## 2. 產品目標

這個產品要解決的問題是：

### 2.1 Prompt 太長，難以管理

圖片 Prompt 往往包含人物、表情、姿態、服裝、場景、光線、鏡頭、風格、構圖、質感等大量信息。全部寫在一大段文字裡，很難重用，也很難穩定復現。

本產品要把長 Prompt 拆成一個個小模塊。

### 2.2 Prompt 太抽象，不能直觀看效果

很多 Prompt 片段只看文字很難理解，例如：

- Y2K 涂鴉混合媒介
- SNS 編輯自拍風格
- 業餘智能手機快照
- 柔和精靈少女
- 馬克筆涂鴉覆蓋層

如果每個素材都有圖片預覽，使用者可以直接看效果，而不是只讀文字。

### 2.3 好 Prompt 不能沉澱

使用者寫出一條好 Prompt 後，不應該只保存整條文字，而應該可以把它拆成：

- 一個人設素材
- 一個表情素材
- 一個姿態素材
- 一個服裝素材
- 一個場景素材
- 一個影像風格素材
- 一個鏡頭質感素材
- 一個妝容素材

這樣以後可以重新組合，生成更多不同 Prompt。

### 2.4 使用者需要自用，不需要公眾平台功能

這個產品是私人使用，不是面向公眾的平台。因此所有分享、公開素材庫、作者資料、積分、圖片生成、社交展示都不需要。

---

## 3. 產品範圍

### 3.1 這個產品要做什麼

本產品只做以下事情：

1. 管理我的 Prompt 素材。
2. 用圖片預覽方式選擇 Prompt 素材。
3. 按分類組合素材。
4. 把素材自動編譯成完整 Prompt。
5. 支持手動修改完整 Prompt。
6. 支持複製完整 Prompt。
7. 支持保存完整 Prompt 到自用 Gallery。
8. 支持從完整 Prompt 分析並拆解成素材。
9. 支持使用者確認、修改、保存拆解出的素材。
10. 支持尺寸與質量作為 Prompt 參數輸入。

### 3.2 這個產品不做什麼

本產品不包含以下任何內容：

1. 不做 AI 圖像生成。
2. 不接入 AI 圖像模型。
3. 不需要 AI 模型選擇。
4. 不需要輸出格式選擇。
5. 不需要 PNG、JPEG、WebP 選項。
6. 不需要積分。
7. 不需要付費提示。
8. 不需要生成圖片按鈕。
9. 不需要最近生成圖片。
10. 不需要角色一致性工作流。
11. 不需要參考圖工作流。
12. 不需要角色卡。
13. 不需要照片復刻。
14. 不需要結構檢測。
15. 不需要 AI 優化建議。
16. 不需要公開素材庫。
17. 不需要我的素材 / 公開素材庫切換。
18. 不需要分享案例。
19. 不需要作者信息。
20. 不需要社交功能。
21. 不需要公開 Gallery。
22. 不需要任何面向公眾的內容運營功能。

## 4. 核心概念

## 4.1 Prompt 素材

Prompt 素材是本產品最小的內容單位。

一個素材不是普通圖片，也不是普通筆記。它是一個帶有圖片預覽的 Prompt 片段。

每個素材至少包含：

1. 分類
2. 標題
3. 副標題
4. 圖片預覽
5. Prompt 正文
6. Negative Prompt
7. 標籤
8. 備註
9. 創建時間
10. 更新時間

例如一個素材可以是：

- 分類：表情
- 標題：俏皮自信微笑
- 副標題：溫柔甜美看鏡頭
- Prompt 正文：playful confident smile, soft sweet expression, looking directly at camera
- 標籤：俏皮、甜美、看鏡頭、自拍

---

## 4.2 素材分類

素材分類不是普通標籤，而是 Prompt 的結構槽位。

完整分類共 16 類：

1. 人設
2. 表情
3. 姿態
4. 上裝
5. 下裝
6. 鞋履
7. 場景
8. 寫真風格
9. 光影
10. 畫面影響
11. 版式設計
12. 配飾
13. 道具
14. 鏡頭質感
15. 景別
16. 妝容

每一類都對應完整 Prompt 裡的一部分。

---

## 4.3 當前組合

當前組合是使用者正在組裝的一條 Prompt。

它由 16 個分類槽位組成。

每個槽位可以是空的，也可以選中一個或多個素材。

使用者在「看圖選擇」裡選中素材後，素材會進入當前組合。

當前組合會自動編譯成完整 Prompt。

---

## 4.4 完整 Prompt

完整 Prompt 是由當前組合裡所有已選素材按固定順序組合出的文字。

完整 Prompt 可以：

1. 自動生成。
2. 手動編輯。
3. 複製。
4. 保存到自用 Gallery。
5. 再次分析並拆解成素材。

---

## 4.5 自用 Gallery

Gallery 是使用者自己的完整 Prompt 收藏庫。

它不是公開案例庫。

它只保存使用者自己認為值得沉澱的完整 Prompt。

Gallery 裡每個項目是一條完整 Prompt，可以有：

1. 標題
2. 預覽圖
3. 完整 Prompt
4. 尺寸參數
5. 質量參數
6. 標籤
7. 備註
8. 創建時間
9. 更新時間

Gallery 裡的 Prompt 可以重新打開、複製、編輯、拆解成素材，也可以重新套用到當前組合。

---

## 5. 整體頁面結構

產品只有一個主要工作台頁面。

頁面分為 5 個區域：

1. 頂部標題區
2. 創作參數區
3. 當前組合區
4. Prompt 編譯器區
5. 自用 Gallery 區

---

## 6. 頂部標題區

## 6.1 位置

頁面最上方。

## 6.2 內容

頂部標題區包含：

1. 產品名稱
2. 一句簡短說明

## 6.3 建議文案

產品名稱：

> Prompt 視覺化素材工作台

說明文字：

> 用看圖選擇的方式，把人設、表情、姿態、服裝、場景、風格、鏡頭和妝容組合成可複用的完整 Prompt。

## 6.4 不要出現的內容

頂部不需要：

1. 積分
2. 購買積分
3. AI 模型名稱
4. 返回公開工作台
5. 用戶社交信息
6. 生成圖片入口

---

## 7. 創作參數區

## 7.1 功能定位

創作參數區只保留兩種參數：

1. 尺寸
2. 質量

這兩個參數只是作為 Prompt 文字的一部分，不用於真正生成圖片。

---

## 7.2 尺寸參數

尺寸使用下拉選單。

選項包括：

1. 自動｜由使用場景決定尺寸
2. 1:1｜1024×1024｜方圖
3. 3:2｜1536×1024｜橫圖
4. 2:3｜1024×1536｜竪圖
5. 1:1｜2048×2048｜2K 方圖
6. 3:2｜2016×1344｜2K 橫圖
7. 2:3｜1344×2016｜2K 竪圖
8. 16:9｜2048×1152｜2K 橫圖
9. 9:16｜1152×2048｜2K 手機竪圖
10. 1:1｜2880×2880｜4K 方圖
11. 3:2｜3456×2304｜4K 橫圖
12. 2:3｜2304×3456｜4K 竪圖
13. 16:9｜3840×2160｜4K 橫圖
14. 9:16｜2160×3840｜4K 手機竪圖

## 7.3 尺寸參數的行為

使用者選擇尺寸後：

1. 頁面保存當前尺寸選擇。
2. Prompt 編譯器把尺寸寫入完整 Prompt 的參數區。
3. Gallery 保存時也保存該尺寸參數。

---

## 7.4 質量參數

質量使用按鈕組。

選項包括：

1. 自動
2. 高
3. 中
4. 低

## 7.5 質量參數的行為

使用者選擇質量後：

1. 頁面保存當前質量選擇。
2. Prompt 編譯器把質量寫入完整 Prompt 的參數區。
3. Gallery 保存時也保存該質量參數。

---

## 8. 當前組合區

## 8.1 功能定位

當前組合區是整個產品的核心操作區。

它讓使用者清楚看到目前 Prompt 已經選了哪些素材，還缺哪些素材。

---

## 8.2 區域標題

標題：

> 當前組合

標題旁顯示：

> 已啟用 X 個素材

其中 X 是目前已選素材數量。

---

## 8.3 大圖選擇按鈕

當前組合區右上角有一個主要按鈕：

> 大圖選擇

按鈕作用：

打開大圖選擇器，讓使用者集中查看並選擇所有分類素材。

---

## 8.4 16 個素材槽位

當前組合區必須顯示 16 個槽位。

順序如下：

1. 人設
2. 表情
3. 姿態
4. 上裝
5. 下裝
6. 鞋履
7. 場景
8. 寫真風格
9. 光影
10. 畫面影響
11. 版式設計
12. 配飾
13. 道具
14. 鏡頭質感
15. 景別
16. 妝容

---

## 8.5 空槽位顯示

如果某個分類尚未選擇素材，槽位顯示：

1. 分類名稱
2. 提示文字

例如：

> 上裝  
> 點擊選擇上裝

---

## 8.6 已選槽位顯示

如果某個分類已選素材，槽位顯示：

1. 分類名稱
2. 素材預覽圖，如果有
3. 素材標題
4. 素材副標題

例如：

> 表情  
> 俏皮自信微笑  
> 溫柔甜美看鏡頭

---

## 8.7 點擊槽位的行為

點擊某個槽位後：

1. 打開大圖選擇器。
2. 大圖選擇器自動切換到該分類。
3. 使用者可以在該分類下選擇素材。
4. 選中後該素材寫入當前組合。
5. Prompt 編譯器自動更新。

---

## 8.8 已選素材的取消

使用者應該可以取消某個槽位的選擇。

取消後：

1. 該槽位變回空槽位。
2. 已啟用素材數量減少。
3. 完整 Prompt 自動重新編譯。

---

## 9. 大圖選擇器

## 9.1 功能定位

大圖選擇器是最重要的功能。

它不是普通素材列表，而是 Prompt 素材的視覺化選擇工作台。

使用者通過它看圖挑選 Prompt 素材。

---

## 9.2 打開方式

大圖選擇器可以由兩種方式打開：

1. 點擊「大圖選擇」按鈕。
2. 點擊當前組合裡任意一個分類槽位。

---

## 9.3 大圖選擇器佈局

大圖選擇器以彈窗形式出現。

彈窗內從上到下分為：

1. 標題欄
2. 搜索與導入工具欄
3. 分類切換欄
4. 標籤篩選欄
5. 素材卡片網格
6. 底部當前選擇摘要

---

## 9.4 標題欄

標題：

> 大圖選擇

右上角有關閉按鈕。

點擊關閉後彈窗消失，但已選內容保留。

---

## 9.5 搜索與導入工具欄

工具欄包含：

1. 搜索框
2. 粘貼 Prompt 導入按鈕

---

## 9.6 搜索框

搜索框用於搜索我的素材。

搜索範圍包括：

1. 素材標題
2. 素材副標題
3. 素材 Prompt 正文
4. 素材標籤
5. 素材備註

搜索結果即時更新素材卡片網格。

---

## 9.7 粘貼 Prompt 導入按鈕

按鈕文字：

> 粘貼 Prompt 導入

點擊後打開「粘貼 Prompt 導入素材」彈窗。

這個功能用於把一條完整 Prompt 分析並拆成多個素材。

---

## 9.8 分類切換欄

分類切換欄顯示全部 16 個分類。

分類順序如下：

1. 人設
2. 表情
3. 姿態
4. 上裝
5. 下裝
6. 鞋履
7. 場景
8. 寫真風格
9. 光影
10. 畫面影響
11. 版式設計
12. 配飾
13. 道具
14. 鏡頭質感
15. 景別
16. 妝容

點擊分類後：

1. 當前分類切換。
2. 素材卡片網格只顯示該分類素材。
3. 標籤篩選欄更新為該分類下的標籤。

---

## 9.9 標籤篩選欄

標籤篩選欄顯示當前分類下已有的標籤。

第一個選項固定為：

> 全部

點擊某個標籤後：

1. 素材卡片網格只顯示帶有該標籤的素材。
2. 搜索條件與標籤條件可以同時生效。

---

## 9.10 素材卡片網格

素材卡片以網格方式排列。

每張卡片代表一個 Prompt 素材。

每張素材卡必須顯示：

1. 圖片預覽
2. 分類名稱
3. 標題
4. 副標題
5. 標籤
6. Prompt 摘要
7. 選中狀態

---

## 9.11 素材卡片圖片預覽

圖片預覽是必需的。

它用來讓使用者直觀看到這個 Prompt 素材大概會帶來什麼視覺效果。

如果素材暫時沒有圖片預覽，卡片仍然可以存在，但必須顯示一個清楚的空預覽占位。

空預覽占位應提示：

> 尚無預覽圖

---

## 9.12 素材卡片選中狀態

當素材已經被加入當前組合時，素材卡應顯示明顯的已選狀態。

已選狀態至少包括：

1. 邊框高亮
2. 勾選標記
3. 已選文字提示

---

## 9.13 點擊素材卡的行為

點擊素材卡後：

1. 該素材加入當前組合。
2. 如果該分類是單選分類，替換同分類原有素材。
3. 如果該分類是多選分類，加入該分類已選列表。
4. 當前組合區同步更新。
5. 完整 Prompt 自動重新編譯。

---

## 9.14 單選分類

以下分類默認為單選：

1. 人設
2. 表情
3. 姿態
4. 上裝
5. 下裝
6. 鞋履
7. 場景
8. 寫真風格
9. 光影
10. 鏡頭質感
11. 景別
12. 妝容

單選分類同一時間只能選一個素材。

---

## 9.15 多選分類

以下分類默認為多選：

1. 畫面影響
2. 版式設計
3. 配飾
4. 道具

多選分類同一時間可以選多個素材。

這些分類通常是可疊加的元素，例如：

1. 涂鴉覆蓋層
2. 貼紙
3. 裝飾圖形
4. 手持道具
5. 首飾
6. 包
7. 眼鏡

---

## 9.16 素材卡操作

每張素材卡應該有以下操作：

1. 選擇
2. 取消選擇
3. 查看完整 Prompt
4. 編輯素材
5. 複製素材 Prompt
6. 刪除素材

---

## 9.17 底部當前選擇摘要

大圖選擇器底部顯示目前已選分類摘要。

例如：

> 已選：人設、表情、姿態、場景、寫真風格、鏡頭質感

底部可以包含：

1. 清空當前分類
2. 清空全部
3. 完成

---

## 10. 粘貼 Prompt 導入素材

## 10.1 功能定位

這個功能是把完整 Prompt 變成素材庫的入口。

使用者把一條完整 Prompt 貼進去，系統分析它，把它拆成多個可保存的素材。

---

## 10.2 打開方式

在大圖選擇器裡點擊：

> 粘貼 Prompt 導入

---

## 10.3 導入彈窗內容

彈窗標題：

> 粘貼 Prompt 導入素材

說明文字：

> 粘貼一條完整圖片 Prompt，系統會分析內容，拆解成可複用的分類素材。拆解後你可以逐項修改，再保存到我的素材庫。

彈窗包含：

1. Prompt 輸入框
2. 取消按鈕
3. 分析並拆成素材按鈕

---

## 10.4 Prompt 輸入框

輸入框是多行文本框。

使用者可以貼入完整 Prompt。

輸入框應允許長文本。

---

## 10.5 取消按鈕

按鈕文字：

> 取消

點擊後關閉導入彈窗，不保存任何內容。

---

## 10.6 分析並拆成素材按鈕

按鈕文字：

> 分析並拆成素材

點擊後：

1. 系統讀取輸入框裡的完整 Prompt。
2. 系統分析 Prompt 的語義結構。
3. 系統嘗試拆分出多個素材項。
4. 系統打開「確認拆解素材」彈窗。

---

## 11. 確認拆解素材

## 11.1 功能定位

這一步是人工確認步驟。

系統拆解出來的素材不應該直接寫入素材庫。

使用者必須能逐項檢查、修改、刪除、保存。

---

## 11.9 副標題輸入框

副標題是素材的簡短說明。

Prompt 正文是素材最重要的內容。

這段文字會在選中素材後被加入完整 Prompt。

使用者必須可以自由修改。

---

## 11.11 Negative Prompt 輸入框

Negative Prompt 用於記錄該素材需要避免的內容。

可以留空。

如果不使用 Negative Prompt，仍然保留欄位，方便以後需要時使用。

---

## 11.12 標籤輸入框

標籤用逗號分隔。

例如：

> 自拍, Y2K, 室內, 柔光

標籤用於大圖選擇器的篩選與搜索。

## 11.13 備註輸入框

1. 這個素材適合什麼風格

3. 這個素材的效果是否穩定
4. 這個素材來自哪一條 Prompt

刪除只影響本次拆解結果，不影響已保存素材庫。

## 12. Prompt 編譯器

## 12.1 功能定位

Prompt 編譯器負責把當前組合轉換成完整 Prompt。

它是文本輸出區，不是 AI 優化工具。

---

## 12.2 位置

Prompt 編譯器位於當前組合區下方。

---

## 12.3 區域標題

標題：

> Prompt 編譯器

---

## 12.4 編譯器內容

Prompt 編譯器包含：

1. 完整 Prompt 文本框
2. 自動 / 自定義模式切換
3. 複製 Prompt 按鈕
4. 重置按鈕
5. 保存到 Gallery 按鈕

---

## 12.5 自動模式

自動模式下：

1. 完整 Prompt 由當前組合自動生成。
2. 使用者選擇或取消素材後，完整 Prompt 會立即更新。
3. 使用者不能直接改寫自動生成的文字，或者修改後會自動切換到自定義模式。

---

## 12.6 自定義模式

自定義模式下：

1. 使用者可以自由修改完整 Prompt。
2. 修改後不會被素材選擇自動覆蓋，除非使用者明確切回自動模式。
3. 自定義 Prompt 也可以保存到 Gallery。

---

## 12.7 複製 Prompt 按鈕

> 複製 Prompt

點擊後，完整 Prompt 複製到剪貼簿。

複製成功後顯示短暫提示：

> 已複製

---

## 12.8 重置按鈕

按鈕文字：

> 重置

點擊後：

1. 清空當前組合。
2. 清空完整 Prompt。
3. 尺寸與質量可保留，也可提供二次確認是否一起重置。

---

## 12.9 保存到 Gallery 按鈕

## 17. 使用者完整操作流程

## 17.1 從素材組裝 Prompt

1. 使用者打開工作台。
2. 使用者選擇尺寸。
3. 使用者選擇質量。
4. 使用者查看當前組合。
5. 使用者點擊「大圖選擇」。
6. 系統打開大圖選擇器。
7. 使用者切換到「人設」分類。
8. 使用者看圖選擇一個人設素材。
9. 使用者切換到「表情」分類。
10. 使用者看圖選擇一個表情素材。
11. 使用者繼續選擇姿態、服裝、場景、風格、光影、鏡頭、妝容等素材。
12. 系統把所有已選素材同步到當前組合。
13. 系統自動編譯完整 Prompt。
14. 使用者檢查完整 Prompt。
15. 使用者複製 Prompt。
16. 使用者可以保存到 Gallery。

---

## 17.2 從完整 Prompt 建立素材

1. 使用者打開大圖選擇器。
2. 使用者點擊「粘貼 Prompt 導入」。
3. 系統打開導入彈窗。
4. 使用者貼入完整 Prompt。
5. 使用者點擊「分析並拆成素材」。
6. 系統拆解 Prompt。
7. 系統打開「確認拆解素材」彈窗。
8. 使用者查看來源 Prompt。
9. 使用者逐項檢查拆解出的素材。
10. 使用者修改分類、標題、副標題、Prompt、標籤、備註。
11. 使用者刪掉不需要的拆解項。
12. 使用者保存單項素材，或批量保存選中素材。
13. 素材被寫入我的素材庫。
14. 使用者回到大圖選擇器。
15. 新素材出現在對應分類下，可以立即被選用。

---

## 17.3 從 Gallery 再利用 Prompt

1. 使用者打開 Gallery。
2. 使用者搜索或篩選已有 Prompt。
3. 使用者打開某個 Gallery 項目。
4. 使用者可以複製完整 Prompt。
5. 使用者可以把它套用到當前組合。
6. 使用者可以把它拆解成素材。
7. 使用者可以編輯後另存為新 Gallery 項目。

## 19. 最終產品判斷標準

這個產品做對了，應該滿足以下標準：

1. 使用者不用從零寫 Prompt。
2. 使用者可以通過看圖選擇理解每個 Prompt 素材的效果。
3. 使用者可以把人物、表情、姿態、服裝、場景、風格、光線、鏡頭、妝容分開管理。
4. 使用者可以自由組合素材，快速得到完整 Prompt。
5. 使用者可以把好 Prompt 保存到 Gallery。
6. 使用者可以把好 Prompt 拆解成可重用素材。
7. 所有素材都只屬於使用者自己。
8. 頁面沒有任何圖片生成、模型選擇、積分、公開分享、作者信息、角色一致性、參考圖工作流等雜訊。

## 20. 產品最終形態

最終產品是一個私人 Prompt 工作台。

它的本質不是圖片生成器，而是：

> 視覺化 Prompt 素材管理器 + Prompt 積木組裝器 + 私人 Prompt Gallery。

使用者每天使用它時，只需要做三件事：

1. 看圖挑素材。
2. 組合出完整 Prompt。
3. 保存和沉澱好用的 Prompt 與素材。
